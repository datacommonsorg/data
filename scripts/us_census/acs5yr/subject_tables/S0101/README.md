# Age and Sex
This table provides data on the age and sex distribution of the total population.
It includes counts of individuals across various age groups and by gender, as well as summary measures like median age and dependency ratios.

The data is sliced by the following properties:
- age
- gender

## Notes
- Median age is provided as a `medianValue` with `Years` as the unit.
- Dependency ratios (Age, Old-age, Child) and Sex ratio are scaled by 100 and use `Percent` as the unit.
- `PERCENT IMPUTED`, `PERCENT ALLOCATED`, and various `Annotation` columns are ignored to prevent duplication and noise.
- Specific dependency ratio combinations are mapped to custom DCIDs using overwrites.
